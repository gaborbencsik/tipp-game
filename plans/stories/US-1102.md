# US-1102: Donation config (Revolut URL template)

> Epic: E11 – Támogatás

**Story:**
Mint **bejelentkezett felhasználó**, szeretnék **a donation modalban működő Revolut payment linkekre kattintani**, hogy **ténylegesen támogathassam a projektet**.

---

## Elfogadási kritériumok

- [x] A DonationModal összeg-gombjai aktív, működő külső linkekként jelennek meg
- [x] A linkek URL-je egyetlen env template-ből épül: `VITE_DONATION_URL`
- [x] A template formátuma: `https://revolut.me/<user>?currency={currency}&amount={amount}&note={note}`
- [x] Az összeg-gombok label/értéke hardcoded a `useDonationConfig` composable-ben (nem env-ből):
  - 1000 Ft, 2000 Ft, 4000 Ft
- [x] Az `{amount}` minor units-ban kerül az URL-be (1000 Ft → 100000)
- [x] A `{currency}` → `HUF`, `{note}` → URL-encoded "VB Tippjáték támogatás"
- [x] "Egyéni összeg →" link: a template az `&amount={amount}` rész nélkül (user maga írja be)
- [x] Minden link új tabban nyílik meg (`target="_blank"`, `rel="noopener noreferrer"`)
- [x] **Graceful degradation:** ha `VITE_DONATION_URL` nincs beállítva → DonationButton nem jelenik meg
- [x] A `.env.example` és `docker-compose.yml` tartalmazza a template-et
- [x] Unit tesztek lefedik: URL-generálás, hiányzó env, openAmountUrl

---

## Technikai megjegyzések

- **Nincs backend érintettség** — tisztán frontend feature
- Composable: `useDonationConfig()` — URL template feldolgozás és pre-set összegek
- Típusdefiníció:
  ```typescript
  interface DonationAmount {
    readonly label: string   // "Egy sör 🍺 — 1000 Ft"
    readonly amount: string  // "1000"
    readonly url: string     // összerakott Revolut URL
  }

  interface UseDonationConfigReturn {
    readonly amounts: readonly DonationAmount[]
    readonly openAmountUrl: string | undefined
    readonly isConfigured: boolean
  }
  ```
