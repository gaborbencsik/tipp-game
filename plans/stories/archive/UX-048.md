# UX-048: Match Detail oldalon a tippek CSV exportálása

## Leírás

Mint **admin/csoport-admin/felhasználó**, szeretném, hogy **a Match Detail oldalon egy gombbal letöltessem a jelenleg megjelenített tippeket CSV formátumban**, hogy **személyre szabott elemzéseket, jelentéseket és archiválásokat tudjak készíteni a csoportomban vagy globálisan**.

## Jelenlegi helyzet

A Match Detail oldal (`MatchDetailView.vue`) megjeleníti az összes tippet egy `MatchPredictionsList` komponensben. Az adatok már a kliensen vannak betöltve (`matchPredictions.value` az `api.predictions.forMatch()`-ből), azonban nincs lehetőség a tippek exportálására.

## Elfogadási kritériumok

### Frontend — UI

1. **"Export CSV" gomb a Match Detail oldalon**
   - Helyezkedik: a tipp-lista felett (vagy mellett), olyan gombként, amely jól látható, de nem ütközik az aktív tartalommal
   - Adat-testid: `data-testid="match-predictions-export-csv"`
   - Állapotok:
     - **Enabled** (alapértelmezés): ha van legalább 1 megjelenített tipp
     - **Disabled** (szürke, cursor: not-allowed): ha a tipp lista üres (0 tipp)
   - Icon (opcionális): letöltés ikon (⬇️ vagy `.svg`)
   - Label: magyar szöveg, pl. "Export CSV" vagy "Letöltés CSV-ként"

2. **Gombkattintás viselkedése**
   - Kattintás → `buildPredictionsCsv(matchPredictions.value, match)` helper meghívása
   - CSV string generálása (lásd Technikai megjegyzések)
   - Blob létrehozása `UTF-8 BOM` fejléccel (Excel-kompatibilitás)
   - Anchor (`<a>`) létrehozása + `href="blob:..."` + `download="tippek-<home-abbreviation>-<away-abbreviation>-<YYYY-MM-DD>.csv"`
   - Click + remove (automatikus letöltés)

3. **i18n kulcsok**
   - `matchPredictions.exportCsv.button` = "Export CSV"
   - `matchPredictions.exportCsv.empty` = "(nincs betöltött tipp)" — csak az disabled state-ben jelenik meg tooltipként vagy data attribútumban
   - CSV oszlop fejlécek:
     - `matchPredictions.exportCsv.header.username` = "Felhasználónév"
     - `matchPredictions.exportCsv.header.prediction` = "Tippelt eredmény"
     - `matchPredictions.exportCsv.header.outcome` = "Döntetlen utáni kimenet"
     - `matchPredictions.exportCsv.header.scorer` = "Gólszerző"
     - `matchPredictions.exportCsv.header.points` = "Meccs pont"
     - `matchPredictions.exportCsv.header.scorerBonus` = "Gólszerző bónusz"
     - `matchPredictions.exportCsv.header.supporter` = "Támogató-e"

### Backend — N/A

- Ez egy **kliens-oldali** export — **nincs új backend endpoint szükséges**. Az adat már a kliensen van (a `predictions.forMatch()` összes szükséges mezőt visszaad).

### CSV formátum

**Oszlopok (fix sorrend):**
1. Felhasználónév (`displayName`)
2. Tippelt eredmény (`homeGoals` – `awayGoals`)
3. Döntetlen utáni kimenet (`outcomeAfterDraw`: pl. "Hosszabbítás", "Tizenegyesek", `null` → üresen hagyva)
4. Gólszerző (`scorerPlayerNameSnapshot`: ha `null`, üres cella)
5. Meccs pont (`pointsResult`: ha `null`, "–")
6. Gólszerző bónusz (`scorerBonusPoints`: ha `null` vagy 0, "–")
7. Támogató (`isSupporter`: "Igen" / "Nem")

**CSV escaping (RFC 4180):**
- Vesszőket, idézőjeleket, újsorokat tartalmazó cellák idézőjelbe kerülnek
- Idézőjel duplikálva (`"` → `""`)
- UTF-8 BOM: `﻿` az első sorban (Excel-kompatibilitás, nincs külön sorként)

**Fájlnév:** `tippek-<home-teamnév-rövidítés>-<away-teamnév-rövidítés>-<YYYY-MM-DD>.csv`
- Pl.: `tippek-HU-DE-2026-07-23.csv` (ahol HU/DE a `match.homeTeam.code` / `match.awayTeam.code`)

**Fejléc sor:** magyar nyelvű, az i18n kulcsokból vett szövegek

### Tesztek (Frontend)

1. **Unit teszt: `buildPredictionsCsv` helper (`lib/predictionsCsv.test.ts`)**
   - **Red**: helper nem létezik
   - **Green**: `buildPredictionsCsv(predictions, match) → string`
   - **Tesztek (TDD):**
     - **Üres lista:** `buildPredictionsCsv([], match)` → csak fejléc sor + newline
     - **Egy tipp:** `buildPredictionsCsv([prediction], match)` → fejléc + 1 adat sor
     - **Speciális karakterek:** displayName-ben vessző/idézőjel → escapel (RFC 4180)
     - **Null pontok:** `pointsResult === null` → cella tartalmazza "–" vagy `pointsResult === 0` → "0"
     - **Null gólszerző:** `scorerPlayerNameSnapshot === null` → üres cella
     - **Döntetlen utáni kimenet:** `outcomeAfterDraw === "AfterExtraTime"` → `"Hosszabbítás"` (i18n fordítás vagy fix szöveg)
     - **BOM:** CSV string első 3 bytja az UTF-8 BOM `﻿`

2. **Komponens teszt: `MatchPredictionsList.test.ts` (vagy `MatchDetailView.test.ts`)**
   - Gomb megjelenik és enabled, ha `matchPredictions.length > 0`
   - Gomb disabled + szürke, ha `matchPredictions.length === 0`
   - Gombkattintás → `buildPredictionsCsv` meghívása a helyes paraméterekkel
   - Blob létrehozása + anchor trigger (`click()` mock vagy spy)
   - Letöltés fájlnév ellenőrzése (`download` attribútum)
   - Aria-label/title: "Export CSV" vagy `matchPredictions.exportCsv.button` i18n érték

### Jogosultság

- **Aki látja a tippeket, exportálhat is** — nem kell külön jogosultság. Ha a `matchPredictions` tömb üres/szűrt (pl. a jogosultságok miatt), az üres CSV-t exportál.
- Szenzitív adat (felhasználónevek) — az export szintje ugyanaz, mint a tipp-lista láthatósága.

## Technikai megjegyzések

### CSV helper létrehozása

Új fájl: `packages/frontend/src/lib/predictionsCsv.ts`

```typescript
export function buildPredictionsCsv(predictions: MatchPrediction[], match: Match): string {
  // UTF-8 BOM
  let csv = '﻿';
  
  // Fejléc sor (i18n fordítások vagy fix magyar szöveg)
  const headers = [
    'Felhasználónév',
    'Tippelt eredmény',
    'Döntetlen utáni kimenet',
    'Gólszerző',
    'Meccs pont',
    'Gólszerző bónusz',
    'Támogató-e'
  ];
  csv += headers.map(escapeCSV).join(',') + '\n';
  
  // Adatsorok
  predictions.forEach((p) => {
    const row = [
      p.displayName,
      `${p.homeGoals}-${p.awayGoals}`,
      outcomeLabel(p.outcomeAfterDraw), // i18n vagy fix label
      p.scorerPlayerNameSnapshot ?? '',
      p.pointsResult !== null && p.pointsResult !== undefined ? p.pointsResult : '–',
      p.scorerBonusPoints ? p.scorerBonusPoints : '–',
      p.isSupporter ? 'Igen' : 'Nem'
    ];
    csv += row.map(escapeCSV).join(',') + '\n';
  });
  
  return csv;
}

function escapeCSV(cell: string | number): string {
  const str = String(cell);
  if (str.includes(',') || str.includes('"') || str.includes('\n')) {
    return '"' + str.replace(/"/g, '""') + '"';
  }
  return str;
}

function outcomeLabel(outcome: MatchOutcome | null): string {
  // i18n fordítás vagy fix label
  if (!outcome) return '';
  const labels: Record<MatchOutcome, string> = {
    'AfterExtraTime': 'Hosszabbítás',
    'AfterPenalties': 'Tizenegyesek',
    // ...
  };
  return labels[outcome] ?? '';
}
```

**Import típusok:** `MatchPrediction`, `Match` — ellenőrizd a `packages/frontend/src/types/index.ts`-ben.

### Component-ben a gomb

```vue
<button
  @click="exportToPredictionsCsv"
  :disabled="matchPredictions.length === 0"
  data-testid="match-predictions-export-csv"
  class="px-4 py-2 bg-blue-600 text-white rounded disabled:bg-gray-300 disabled:cursor-not-allowed"
>
  {{ $t('matchPredictions.exportCsv.button') }}
</button>

<script setup>
import { buildPredictionsCsv } from '@/lib/predictionsCsv';

const exportToPredictionsCsv = () => {
  const csv = buildPredictionsCsv(matchPredictions.value, match.value);
  const blob = new Blob([csv], { type: 'text/csv;charset=utf-8;' });
  const link = document.createElement('a');
  link.href = URL.createObjectURL(blob);
  link.download = `tippek-${match.value.homeTeam.code}-${match.value.awayTeam.code}-${new Date().toISOString().split('T')[0]}.csv`;
  link.click();
  URL.revokeObjectURL(link.href);
};
</script>
```

## Kizárások

- **Nem része:** korlátlan tippek exportálása (pl. több meccsre, több csoportra) — ez a story csak az aktuálisan megjelenített `matchPredictions`-t exportálja
- **Nem része:** szerver-oldali CSV generálás — nincs új backend endpoint
- **Nem része:** haladó CSV-szűrés (oszlop választás, szortírozás) — fix oszlop-sorrend, az összes tipp exportálódik
- **Nem része:** szerverről lekért export (auditálás, audit log) — csak kliens-oldali adat
