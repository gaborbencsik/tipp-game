# US-1101: Donation gomb és pop-up

> Epic: E11 – Támogatás

**Story:**
Mint **bejelentkezett felhasználó**, szeretnék **egy visszafogott "Támogatás" gombot látni a sidebarban**, hogy **egyszerűen tudjam támogatni a projektet, ha szeretném**.

---

## Elfogadási kritériumok

- [x] A bal oldali navigációs sávban (AppLayout sidebar) a nav itemek alatt, az aljára rögzítve megjelenik egy "🍺 Fizess egy sört!" gomb
- [x] A gomb stílusa visszafogott (amber/warm szín), illeszkedik a nav többi eleméhez
- [x] Összecsukott sidebarban csak a 🍺 emoji látható, hover-re tooltip jelenik meg ("Fizess egy sört!")
- [x] Kattintásra egy kisméretű modal jelenik meg (max-w-sm, centered), tartalma:
  - Headline: "Tetszik a játék? 🍺"
  - Body: "Ezt a tippjátékot hobbiból, a közösség öröméért készítem. Ha neked is örömet okoz, egy kisebb támogatással te is hozzájárulhatsz a működéséhez."
  - **Összeg gombok (horizontális sor, 3 db):**
    - "Egy sör 🍺 — 1000 Ft" (outlined/ghost stílus)
    - "Egy kör 🍻 — 2000 Ft" (kiemelt, filled amber stílus — vizuálisan ajánlott opció)
    - "VIP páholy 🏟️ — 4000 Ft" (outlined/ghost stílus)
  - **Alatta:** "Egyéni összeg →" szöveglink (secondary, nem gomb) — nyitott összegű Revolut link
  - Dismiss link: "Köszi, most nem" (szöveglink, nem gomb — bezárja a modalt)
  - **Alatta kis betűvel:** "Ne kérdezd többé" link — permanent suppress (localStorage flag)
- [x] A középső gomb ("Egy kör") vizuálisan kiemelt (filled amber/primary), a másik kettő outlined — anchoring effektus
- [x] Minden összeg-gomb és az egyéni link is külső linkként működik (`target="_blank"`, `rel="noopener noreferrer"`)
- [x] Ha a donation config nincs beállítva (`VITE_DONATION_URL` hiányzik), a gomb nem jelenik meg
- [x] A gomb nem tolakodó: nem nyílik meg automatikusan, csak explicit kattintásra (az admin-triggered popup külön story: US-1103)
- [x] A modal ESC-re és backdrop kattintásra is bezárul
- [x] A gomb `aria-label="Támogasd a projektet"` attribútumot kap (accessibility, különösen collapsed állapotban)
- [x] A vizuális stílus illeszkedik az app többi eleméhez (Tailwind, kis méret, warm/amber tónus)

---

## Technikai megjegyzések

**Komponens struktúra:**
- `DonationButton.vue` — sidebar gomb + collapsed tooltip
- `DonationModal.vue` — modal tartalom (összeg-gombok + egyéni link + dismiss)
  - Az összeg-gombok hardcoded a composable-ben, URL-jük a `VITE_DONATION_URL` template-ből épül
  - A középső elemet (index 1) kiemelt stílussal jeleníti meg
- `useDonationConfig.ts` — URL template feldolgozás (`{currency}` → HUF, `{amount}` → minor units, `{note}` → "VB Tippjáték támogatás")

**URL template:** `https://revolut.me/gaborbencsik?currency={currency}&amount={amount}&note={note}`
- Az összeg minor units-ban megy (1000 Ft → `amount=100000`)
- A note URL-encoded: `VB%20Tippj%C3%A1t%C3%A9k%20t%C3%A1mogat%C3%A1s`
