# US-1402: Játékos-választó ligaszűrés és szabad szöveges fallback

## Leírás

Mint **felhasználó**, szeretném, hogy a speciális tipp játékos-választó **csak az adott liga játékosait kínálja fel** (név és csapat szerint kereshető), és ha nincs játékosadat a DB-ben az adott ligához, akkor **szabad szöveges bevitelt** kapjak helyette.

## Jelenlegi helyzet

- `PlayerSelectCombobox.vue` betölti az **összes** playert (`GET /api/players`), nincs liga/csoport filter
- A backend `getPlayers(teamId?)` támogat `teamId` szűrést, de a route nem adja át
- Ha nincs player a DB-ben → üres dropdown, "Nincs ilyen játékos" felirat → a user nem tud tippelni
- Nincs fallback szabad szöveges mód

## Elfogadási kritériumok

### Backend

- [ ] `GET /api/players` kap egy opcionális `leagueId` query paramétert
- [ ] Ha `leagueId` megadva → csak azokat a playereket adja vissza, akiknek a csapata (`teams`) az adott ligához tartozik (a `teams` és liga kapcsolat alapján)
- [ ] Új endpoint: `GET /api/players/count?leagueId=X` → `{ count: number }` — a frontend ezzel dönti el, van-e player adat (vagy a `GET /api/players?leagueId=X` response length alapján)
- [ ] A `player_select` típusú special prediction validáció:
  - Ha vannak játékosok a ligában → `answer` kötelezően valid player UUID
  - Ha nincs játékos a ligában → `answer` bármilyen non-empty string (max 200 karakter)

### Frontend — PlayerSelectCombobox bővítés

- [ ] Új prop: `leagueId?: string` — ha megadva, a `GET /api/players?leagueId=X` endpointot hívja
- [ ] Ha a liga-szűrt player lista **nem üres** → a jelenlegi keresős dropdown működik (liga játékosai, név + csapat)
- [ ] Ha a liga-szűrt player lista **üres** → automatikusan szabad szöveges módra vált:
  - A combobox eltűnik, helyette egy sima text input jelenik meg
  - Placeholder: "Írd be a játékos nevét"
  - Max 200 karakter
  - Az `answer` ilyenkor plain text (nem UUID)
- [ ] A keresős dropdown a korábbiakhoz hasonlóan mutatja: `Játékosnév (Csapat)` formátumban
- [ ] A keresés továbbra is diakritikaérzékeny (normalizált összehasonlítás)

### UX részletek

- [ ] A dropdown maximum 50 találatot mutat (már így van)
- [ ] Ha nincs találat keresésre (de vannak játékosok a ligában) → "Nincs ilyen játékos" felirat (megmarad)
- [ ] A szabad szöveges mód vizuálisan ne különbözzön drasztikusan — maradjon a jelenlegi input stílus
- [ ] Ha a user már korábban UUID-t mentett és azóta fallback módra váltott (edge case: player adat törölve) → az input mutatja a korábban mentett szöveget (a `displayName` fallback-je az ID-ről szövegre)

## Technikai megjegyzések

### Liga → csapatok → játékosok kapcsolat

A jelenlegi schema alapján:
- `groups` → `group_leagues` → `leagues` (csoport ligái)
- `teams` van, de a teams-leagues kapcsolatot ellenőrizni kell — ha nincs explicit FK, a `special_prediction_types.group_id` → `group.leagues` → teams kapcsolat kell

**Vizsgálandó:** Hogyan kötődnek a teams egy ligához? Ha nincs direkt FK, szükség lehet egy `league_teams` junction tábla-ra, vagy a meglévő `matches` tábla alapján kikövetkeztethető (a meccsben szereplő csapatok = liga csapatai).

### Alternatív approach (ha nincs league→teams direkt FK)

A `matches` tábla `league_id` + `home_team_id` / `away_team_id` mezőiből kinyerhető a liga összes csapata:
```sql
SELECT DISTINCT home_team_id AS team_id FROM matches WHERE league_id = ?
UNION
SELECT DISTINCT away_team_id FROM matches WHERE league_id = ?
```
Innen a `players.team_id IN (...)` szűrés adja a liga játékosait.

### Validáció switch logika (backend)

```typescript
if (type.inputType === 'player_select') {
  const leagueId = getLeagueIdForGroup(type.groupId)
  const playerCount = await countPlayersByLeague(leagueId)

  if (playerCount > 0) {
    // Strict mode: answer must be valid player UUID in this league
    const player = await getPlayerById(input.answer)
    if (!player || !isPlayerInLeague(player, leagueId)) {
      throw new AppError(422, 'Invalid player selection')
    }
  } else {
    // Free text mode: any non-empty string
    if (!input.answer.trim() || input.answer.length > 200) {
      throw new AppError(422, 'Player name must be 1-200 characters')
    }
  }
}
```

### Frontend mode detection

```typescript
const isDropdownMode = computed(() => playersList.value.length > 0)
```

Ha `!isDropdownMode` → renderelj egy sima `<input type="text">` a combobox helyett.
