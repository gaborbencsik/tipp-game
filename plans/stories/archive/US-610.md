# US-610: Mérkőzések oldal liga szűrő

> Epic: E1 – Mérkőzések

**Story:**
Mint **felhasználó**, szeretnék **a mérkőzések oldalon liga szerint szűrni** (Összes / adott liga), hogy **gyorsan megtaláljam a releváns meccseket ha több liga is van a rendszerben**.

**Kontextus:**
Jelenleg nincs frontend liga filter a meccsek oldalon — a backend már támogatja a `?leagueId=X` paramétert. A liga lista a `GET /api/leagues` végpontból elérhető. Ez a story UI-only: egy select dropdown a stage filterek mellé.

**Elfogadási kritériumok:**
- [ ] A mérkőzések oldal tetején a stage filter sor mellett megjelenik egy liga szűrő select
- [ ] Opciók: "Összes liga" (default) + az összes elérhető liga neve
- [ ] Kiválasztás után a meccsek lista azonnal frissül (csak az adott liga meccsei)
- [ ] "Összes liga" → nincs szűrés, minden meccs látszik
- [ ] Ha csak 1 liga létezik, a select nem jelenik meg (felesleges)
- [ ] A kiválasztott liga szűrő megmarad navigáció után is (store state)
- [ ] A stage filter és liga filter együtt működik (kombinálható)
- [ ] Unit teszt: filter változtatás → matchesStore.fetchMatches hívás a megfelelő paraméterrel
