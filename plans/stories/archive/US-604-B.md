# US-604-B: Meghívó kód kezelése (admin)

> Epic: E6 – Csoportok

**Story:**
Mint **csoport admin**, szeretnék **a csoport meghívó kódját újragenerálni vagy deaktiválni**, hogy **megakadályozzam illetéktelenek csatlakozását**.

**Elfogadási kritériumok:**
- [x] `PUT /api/groups/:groupId/invite` — új 8 karakteres kód generálása (régi érvénytelenül), `inviteActive = true`
- [x] `PATCH /api/groups/:groupId/invite` — invite aktív/inaktív togglelása (`inviteActive` mező)
- [x] Csak csoport admin hívhatja
- [x] Frontend: `GroupDetailView` beállítások szekcióban megjelenik a meghívó kód + Másolás gomb + Újragenerálás gomb + Deaktiválás toggle
- [x] Újragenerálás előtt figyelmeztető üzenet: „A régi meghívó link érvénytelen lesz"
- [x] Ha `inviteActive = false`, a GroupsView-ban a meghívó kód „Inaktív" felirattal jelenik meg
- [ ] Unit tesztek: újragenerálás, deaktiválás, jogosultság ellenőrzés

**Komplexitás:** S
**Prioritás:** Must Have

**Státusz:** ✅ Kész
