# US-604-A: Csoport tagkezelés (admin)

> Epic: E6 – Csoportok

**Story:**
Mint **csoport admin**, szeretnék **a csoport tagjait kezelni** — listázni, eltávolítani, admin szerepkört átadni —, hogy **a csoport összetételét én kontrolláljam**.

**Elfogadási kritériumok:**
- [x] `GET /api/groups/:groupId/members` — taglista (csak csoport tagoknak), tartalmazza: userId, displayName, avatarUrl, isAdmin, joinedAt
- [x] `DELETE /api/groups/:groupId/members/:userId` — tag eltávolítása (csak csoport admin, saját magát nem távolíthatja el)
- [x] `PUT /api/groups/:groupId/members/:userId/role` — admin szerep átadása (csak csoport admin, saját magától elveheti ha van másik admin)
- [x] Backend: mindhárom endpoint `authMiddleware` + csoport-admin jogosultság ellenőrzés
- [x] `GroupMember` típus (backend + frontend): `userId`, `displayName`, `avatarUrl`, `isAdmin`, `joinedAt`
- [x] Frontend: `GroupDetailView` kiegészítve „Tagok" tabon — taglista táblázat (avatar, név, szerep badge, csatlakozás dátuma, Eltávolít / Admin átadás gombok)
- [x] Saját sor kiemelve, saját magát eltávolítani nem lehet (gomb disabled)
- [x] Törlés előtt confirm dialog
- [ ] Unit tesztek: taglista, eltávolítás, jogosultság ellenőrzés

**Komplexitás:** S
**Prioritás:** Must Have

**Státusz:** ✅ Kész
