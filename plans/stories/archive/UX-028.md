# UX-028: Torna tippek mobil UX redesign — bracket + group standings polish

## Leírás

Mint **VB tippelő**, szeretném, hogy a **torna tippek felülete (csoport végeredmény + bracket progresszió)** jobban nézzen ki és intuitívabban működjön a mobilon, hogy **élvezhetőbb legyen a tippelés és könnyebben lássam az állapotokat** — aktív körök, lezárt körök, kitöltöttség.

## Jelenlegi helyzet

Az US-945 (GroupStandingsPredictionView + komponensek) és US-946 (BracketProgressionPredictionView + komponensek) MVP megvalósítása **működőképes**, de az UX még nem felel meg teljesen a `docs/superpowers/mockups/group-standings-prediction-v2.html` és `bracket-progression-prediction-v2.html` mockupoknak.

## Elfogadási kritériumok

### Group Standings (US-945 → v2 mockup)

1. **Aktív csoport kiemelése**
   - Az éppen szerkesztett csoport kártyájának szegélye: `border-2 border-blue-400 ring-4 ring-blue-100 shadow-md`
   - Kész csoport (4/4): `border border-emerald-300 shadow-sm` + `✓` zöld checkmark az összefoglalóban
   - Érintetlen csoport (0/4): alapértelmezett szürke border

2. **Pozíció-badge (pos-badge) megjelenítése**
   - Minden pozíció-sorban **bal oldalt** egy kerek 28×28 pixel **pos-badge** számmal (1., 2., 3., 4.)
   - Kitöltött pozíció: `bg-blue-50 color-indigo-600` (szín: `text-indigo-600 bg-indigo-50`)
   - Üres pozíció: `bg-slate-100 color-slate-400` (szín: `text-slate-400 bg-slate-100`)
   - Kész csoport pozíciói: `bg-emerald-50 color-emerald-600`

3. **Pozíciósor styling (PositionTeamDropdown / kitöltött sor)**
   - A pozíciósor szürke háttér: `bg-slate-50` (row wrapper)
   - A dropdown button stílusa: `border border-slate-300 rounded-md px-2.5 py-1.5 bg-white text-left` (lapos, nem prominent)
   - Egy már kiválasztott csapat megjelenítése a dropdown-ban: **kijelölt állapotban is csak** a csapat neve + zászló, mint egy opció; a szülő nem rendereli külön.
   - **Üres pozícióra kattintás** vagy már kitöltött pozíció módosítása: a dropdown lenyílik, az opciók a szabad csapatok.

4. **Best 3rd lockolt kártya (12 csoport előtt)**
   - Szürke `🔒` lakat-ikon + szöveg: "Töltsd ki előbb mind a 12 csoportot"
   - Szín: `bg-slate-100 border-2 border-dashed border-slate-300`
   - Nem interaktív (disabled), csak információ.

5. **Best 3rd grid kártyák (12/12 után)**
   - 3 oszlopos grid: csapatok 3×4 karton-szerű chipek
   - **Kiválasztott chip:** `border-2 border-blue-500 bg-blue-50` + `✓` jobb felül
   - **Üres chip:** `border border-slate-200 bg-white` + hover: `border-blue-300`
   - Alatta: csapat flag + 3-betűs kód + "X csop." footer

6. **Sticky bottom progress bar**
   - Text: `"X / 13 kész"` + `"✓ Mentve"` (vagy save status)
   - Alatta: **13 darab** kis szegmens-bar (12 csoport + 1 best 3rd):
     - Kitöltött: `bg-emerald-500` (100%)
     - Részleges (best 3rd kész, de nem 4/4): `bg-blue-400` (nem ezt mutatjuk, csak teljes vagy nem)
     - Üres: `bg-slate-200`

7. **Info kártya (legfelül)**
   - Kék `💡` icon + szöveg: "Tippeld meg mind a 12 csoport (A–L) sorrendjét. **Bármelyik csoporttal kezdheted.**..."
   - Szín: `bg-blue-50 border border-blue-200`

### Bracket Progression (US-946 → v2 mockup)

8. **Match Card layout**
   - 3 oszlopú grid: `grid-template-columns: 1fr auto 1fr` (bal csapat — VS — jobb csapat)
   - **Match label** (gomb fölött): "Match X" + pill-like badge pl. `"A1 vs B2"` vagy `"M1 ↔ M2"` (slot-nevek vagy match-referenciák)
   - Bal/jobb: `TeamSlotChip` gomb-szerű kártya (~120px széles)

9. **TeamSlotChip styling**
   - Border: `border border-slate-300` (alapértelmezett)
   - Padding: `px-2.5 py-2`
   - Fentebb: kicsi szürke szöveg `text-slate-400 text-xs` — slot ID (pl. "W A1", "M7 győztes")
   - Alatta: csapat flag + 3-betűs kód + slot-ID label
   - **Kiválasztott (winner) state:** `border-2 border-blue-500 bg-blue-500` + fehér szöveg + ✓ jobb felül zöld badge (`bg-green-500`)
   - **Lockolt state:** `border-dashed border-slate-400 bg-slate-100` + szürke szöveg, kurzor `not-allowed`
   - **Hover (nem lockolt, nem winner):** `border-blue-300 bg-blue-50`

10. **Match Row layout per-match**
    - Minden match alatt: `<match-label>` + `<match-row>` grid
    - Távolság mérkőzések között: `gap-y-3` vagy `mb-4`

11. **RoundCard aktív állapot**
    - Az éppen nyitott, szerkeszthető kör: `border-2 border-blue-400 ring-4 ring-blue-100 shadow-md`
    - Lockolt kör (upstream nincs kész): `border-2 border-dashed border-slate-400 bg-slate-100` + ikon `🔒` + szöveg "Tölts ki előbb X előző mérkőzést"
    - Kész kör (mind 16/8/4/2/1 mérkőzés kész): `border border-emerald-300 shadow-sm`

12. **FinalsAndBronzeCard speciális styling**
    - Két mérkőzés (Bronze + Final) egy közös kártya alatt
    - **Bronz** első: fejléc "🥉 Bronz (SF1 vs SF2 vesztesek, auto)" — a pár auto-szinkronizálódik
    - **Döntő** második: fejléc "🏆 Döntő (SF1 vs SF2 győztesek)"
    - Alatta summary: "🥇 Tipped: FRA · 🥈 BRA · 🥉 ARG · 4. GER" (opcionális ha pontok számítottak)

13. **GroupStandingsGate (US-945 nélkül)**
    - Lockolt prompt kártya: `bg-amber-100 border-2 border-amber-400`
    - Icon: 🔒 nagy
    - Szöveg: "Töltsd ki előbb a Csoport végeredmény tippet (12 csoport × 4 helyezés + 4 továbbjutó 3.helyezett)"
    - Gomb: "→ Csoport tipp megnyitása" (deeplink az US-945 nézetbe)

14. **Sticky bottom progress stepper**
    - **Dot-stepper:** 6 kör (Last 32, Last 16, QF, SF, Final, Bronze — vagy összes match egy "super dot")
    - **Alternatíva:** 32 micro-dot (minden match egy pont) — a mockup szerint ez a verzió
    - State: `.dot` (üres), `.dot.partial` (sárga), `.dot.done` (zöld), `.dot.active` (kék, pulsing)
    - Szöveg: `"Last 32: X/16 · ✓ Mentve"` vagy `"↻ Mentés…"`

15. **Read-only nézet (deadline után)**
    - Helyes pick: `border-2 border-green-500 bg-green-50` + `✓✓` dupla pipa
    - Helytelen pick: `border-2 border-red-500 bg-red-50`
    - Tényleges győztes (admin reveal): `border-2 border-dashed border-green-500 bg-white` + floating label "tényleges győztes"
    - Alatta: kicsi szöveg "A tipped: X · Tényleges: Y"

16. **Info kártya (legfelül)**
    - Kék `💡` icon + szöveg: "Minden mérkőzésnél válaszd ki, ki nyer. A párok a **csoport-tippedből** állnak össze. A győztesek automatikusan tovább kerülnek a következő körbe."
    - Szín: `bg-blue-50 border border-blue-200`

### Globális / közös

17. **Animáció / Transition**
    - Accordion header rotate chevron: smooth 200ms
    - Chip-pick highlight: smooth 150ms border + bg transition
    - Progress bar: smooth opacity/width transition

18. **Color palette (Tailwind v4)**
    - Aktív/nyitott: `blue-400`, `blue-50`, `blue-600`
    - Kész: `emerald-500`, `emerald-300`, `emerald-50`
    - Lockolt: `slate-300`, `slate-100`, `slate-400`
    - Mentve: `emerald-500` / `emerald-600`
    - Hibás/régi: `red-500`, `red-50`
    - Pozícióbadge: `indigo-600 bg-indigo-50` (alapértelmezett), `emerald-600 bg-emerald-50` (kész)

## Technikai megjegyzések

### Frontend komponensek átírása

**GroupStandingsPredictionView.vue:**
- `GroupStandingCard.vue` aktív státusz: `open` prop + Tailwind class binding körül `border-2 border-blue-400` stb.
- Info kártya: standard blue pill layout (utánozza a mockupot)
- Sticky bottom: **13-elemű segment-bar** helyett szegmens (11 szürke + 1-2 kitöltött)

**GroupStandingCard.vue:**
- Header: kész-badge (`✓` + zöld) vs. progress badge (`X/4` + szürke)
- Expanded: 4 sor `PositionTeamDropdown` × `pos-badge` + szürke háttér row wrapper

**PositionTeamDropdown.vue:**
- Button styling: `border border-slate-300 px-2.5 py-1.5` (lapos, nem kiemelkedő)
- Dropdown opciók: szabad csapatok listája (smart filter működik már)
- Az aktuális csapat **nem** jelenik meg külön "kiválasztott" cell-ként; csak a dropdown mutatja

**Best3rdPicker.vue:**
- Lockolt state: amber/slate kerettel + lakat ikon
- Unlockolt grid: 3×4 chip layout
- Kiválasztott chip: `border-2 border-blue-500 bg-blue-50` + ✓ jobb felül

**BracketProgressionPredictionView.vue:**
- Info kártya: blue pill (mockup szerint)
- `RoundCard` wrapper: aktív vs. lockolt vs. kész styling

**RoundCard.vue:**
- Header aktív: `border-2 border-blue-400 ring-4 ring-blue-100 shadow-md`
- Header lockolt: `border-2 border-dashed border-slate-400 bg-slate-100`
- Expanded: az összes `MatchCard` szóköz-rendezés `space-y-3`

**MatchCard.vue:**
- 3-oszlop grid: `grid-template-columns: 1fr auto 1fr`
- Match label: "Match X" + pill-like label pill (slot-Info / match-ref)
- `TeamSlotChip` × 2 + VS középen

**TeamSlotChip.vue:**
- Alapszegély: `border border-slate-300`
- Winner state: `border-2 border-blue-500 bg-blue-500 text-white shadow`
- Lockolt state: `border-dashed border-slate-400 bg-slate-100 text-slate-400 cursor-not-allowed`
- Hover: `border-blue-300 bg-blue-50`
- Layout: flex column, gap-2, fentebb slot-ID (kicsi szürke), alatta flag + 3-letter code

**FinalsAndBronzeCard.vue:**
- Bronz mérkőzés **előbb** (Bronze játszódik korábban a turnán)
- Final mérkőzés után
- Opcionális summary sor (zöld háttér, ha pontok rendelkezésre állnak)

**GroupStandingsGate.vue:**
- Amber 🔒 kártya: "Töltsd ki előbb a csoport tippet"
- Gomb: deeplink az US-945 tab-hoz

### Sticky bottom progress (BracketProgressionPicker)

- **Variáció 1 (mockup szerint):** 32 micro-dot (vagy 6 körönként csoportosított dot)
- **Variáció 2:** "Last 32: X/16 · Last 16: Y/8 · QF: Z/4 ..." (szöveg-alapú, kompaktabb)
- **Javaslat:** micro-dot stepper mint a mockupban — rövid, vizuális

### Jelenlegi komponensek, amelyek nem módosulnak

- `PositionTeamDropdown` működéslogikája (smart filter, none-selection handling) — csak styling
- `BracketProgressionStore` és `GroupStandingsStore` — nincsenek UI/logika változások
- Validáció, API hívások — nincsenek változások
- i18n kulcsok — szöveg felülvizsgálat, de a szerkezet nem változik

## Kizárások

- **Drag-and-drop csaporrendezés** — out of scope; a pozíciódrpdown mechanika marad.
- **Klasszikus bracket-ábra desktop view** — out of scope; mobil-szerű accordion marad desktop-on is.
- **Animált progressbar** (pl. SVG animated circle) — out of scope; alapszínű bar marad.
- **Match-szintű pontozás** — külön story.
- **Admin edit UI** — külön story.
- **Offline cache** — out of scope.
- **Új API endpoint** — nincsen; csak Tailwind styling.
