Design jegyzetek

  • Színek a meglévő palettából: blue-600 primary, gray-300/200 secondary border, red/yellow/green-50 accent háttér.
  • Modal kártya: rounded-2xl shadow-xl — ugyanaz mint a jelenlegi OnboardingOverlay.vue.
  • Mobile-on a kártya helyén inset-0 full-screen sheet (sticky header + footer). Tablet: max-w-lg, desktop: max-w-2xl.
  • Csak továbbléptető CTA-k — minden középső gomb előrevisz, hogy a user végignézze az összes lépést.
  • Progress: 5 dot + "n/5" számozás. Az utolsó dot kicsit nagyobb (ring-2) — jelzi a hub/záró lépést.
  • Animáció (a mockupban statikus): hero ikonok stagger 80–100ms, odds bar width 0→target 600ms, ×2 badge scale-pop.
  • Reduced-motion: minden transition ≤100ms opacity fade, nincs konfetti.
